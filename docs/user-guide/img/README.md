# User-guide screenshots

Drop PNG screenshots here using the **exact filenames** below. The interactive
guide (`../index.html`) references them by name and shows a labelled placeholder
until the file exists — so you can add them one at a time, in any order, and
they appear automatically.

> Tip: capture at a consistent width (e.g. a ~1280px-wide browser window),
> light theme, with the **Equipment Inventory** example register loaded. Crop to
> the relevant panel; annotations/callouts are welcome but optional.

| Filename | What it should show |
|----------|---------------------|
| `01-app-nav.png` | DataForms open in the Nextcloud top bar, with the register navigation list on the left. |
| `02-register-list.png` | The register dashboard — cards for each register, with record counts and favourite stars. |
| `03-table-overview.png` | The Equipment Inventory **records table**: toolbar on top, pinned header, rows, row actions on the right. |
| `04-search.png` | The table with a term typed into **Search** and the results narrowed. |
| `05-filter.png` | The **Filter** bar open with one or two conditions (e.g. Status is *On loan*). |
| `06-sort.png` | The table sorted by a column (e.g. *Due back*), with the sort arrow visible in the header. |
| `07-columns.png` | The **⋯ More → Columns** picker with some fields ticked/unticked. |
| `08-inline-edit.png` | A cell being edited inline after a double-click (e.g. the *Status* dropdown in a row). |
| `09-new-record.png` | The **New record** data-entry form for an equipment item (relation picker for *Category*). |
| `10-record-detail.png` | A record's **detail** view with the **History** timeline expanded at the bottom. |
| `11-csv.png` | The **Import CSV** dialog (format help, template download, per-row results). |
| `12-saved-views.png` | Saving a view and/or the saved-view selector in the toolbar. |
| `13-share.png` | The **Share** dialog: user/group search + Read/Write/Manage role picker. |
| `14-fields.png` | The **Fields** schema editor (managers): the list of fields and the add/edit field dialog. |
| `15-form-builder.png` | The drag-and-drop **Forms** builder: palette of fields + the canvas with sections. |
| `16-rules.png` | The **Rules** builder with an example rule (e.g. require *Assigned to* when *Status* is *On loan*). |
| `17-automations.png` | The **Automations** builder showing a When / Only if / Then configuration. |

Filenames are referenced in `../index.html`. If you rename one, update it there too.
